# Data Warehousing with Hadoop and Hive

This module focuses on building a foundation in distributed data storage and data warehousing using the Hadoop ecosystem.

The main objective is to understand how large-scale data can be stored, managed, and queried efficiently using Hadoop and Hive.

## Technologies Used

- Hadoop (HDFS)
- Hive
- SQL
- Docker

## Topics Covered

- Hadoop Distributed File System (HDFS)
- Data storage in distributed systems
- Hive architecture
- Creating Hive tables
- External vs managed tables
- Partitioning and bucketing
- Loading data into Hive tables
- Querying data using HiveQL

## Files in This Folder

- Hive table creation examples
- Data loading scripts
- HiveQL queries
- Practice exercises and case studies

## Goal

The goal of this module is to understand how data warehouses are built on top of distributed storage systems and how Hive provides a SQL-like interface for large-scale data processing.
