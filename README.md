# data-engineering-labs

This repository contains hands-on experiments and practice related to modern data engineering technologies.

The goal of this repository is to build practical understanding of distributed data systems, ETL pipelines, and cloud-based data infrastructure.

## Technologies Covered

- Python
- SQL
- Hadoop
- Hive
- Apache Spark
- ETL Pipelines
- AWS (Lambda, Glue, Redshift)
- Snowflake
- MongoDB
- Azure Data Factory
- Databricks
- Kafka
